# Candidate Management App

A simple Candidate Management application built with **Next.js** that allows users to **add, list, and delete candidates**. This project demonstrates **basic CRUD functionality**, component-based architecture, and clean UI handling.

---

Features
- List all candidates (Name, Email, Role)
- Add a new candidate via a form
- Delete a candidate
- Data stored in local state or localStorage
- Basic form validation to ensure correct input
- Reusable components and clean folder structure

---

---

Tech Stack

- Framework:Next.js (App Router)
- Styling: Tailwind CSS 
- State Management: React useState / localStorage
- Language: TypeScript
